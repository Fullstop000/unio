# Security policy

## Supported versions

| Version | Security support |
| --- | --- |
| v0.2.x | Supported |
| v0.1.x | Not supported |

## Report a vulnerability

Do not open a public issue for a suspected vulnerability. Email
`fullstop1005@gmail.com` with:

- the affected unio version or commit;
- the selected runtime and exact CLI version;
- the impact and required preconditions;
- a minimal reproducer; and
- any suggested mitigation.

Do not attach credentials or unredacted `Session.Raw` data. If logs are needed,
remove prompts, source code, paths, environment values, tool input/output, and
provider tokens first.

For non-sensitive bugs and compatibility reports, use [GitHub
Issues](https://github.com/Fullstop000/unio/issues) and follow the information
checklist in [CONTRIBUTING.md](CONTRIBUTING.md#reporting-bugs).
